This root is a flat Next.js 16 project (no nested workspaces) whose children are organized under `src/` and communicate through shared imports rather than inter-process contracts. Cross-child wiring happens at three levels: (1) `next.config.mjs` enables React Compiler and whitelists remote image hosts; (2) `jsconfig.json` maps the `@/*` alias to `./src/*`, giving every child module — including `content_lib` — a single import surface for components, hooks, and stores; (3) ESLint (`eslint.config.mjs`) extends `eslint-config-next/core-web-vitals` uniformly across all children. The only runtime cross-cutting concern is the `content_lib`'s localStorage-backed bilingual store, which other children consume as a plain JS module to resolve Arabic/English fields at render time.
